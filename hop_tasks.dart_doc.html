Task createDartDocTask(delayedLibraryList, {String targetBranch: 'gh-pages', String packageDir: 'packages/', Iterable<String> excludeLibs, bool linkApi: false, Func2 postBuild}) #

targetBranch the Git branch that will contain the generated docs. If the branch doesn't exist, it will be created. Default: gh-pages

packageDir the package directory for the current project. Default: packages/

delayedLibraryList a [List] mapping to paths to libraries or some combinations of Future or Function values that return a [List].

postBuild A Function to call before content is committed. It has the signature Future postBuild(TaskLogger logger, String tempDocPath). Use this if you want to modify the doc output.

Task createDartDocTask(dynamic delayedLibraryList, {
 String targetBranch: 'gh-pages',
 String packageDir: 'packages/',
 Iterable<String> excludeLibs,
 bool linkApi: false,
 Func2<TaskContext, String, Future> postBuild
 }) {
 requireArgumentNotNull(targetBranch, 'targetBranch');
 requireArgumentNotNull(packageDir, 'packageDir');

 return new Task((ctx) {
   targetBranch = ctx.arguments[_targetBranchArg];

   return _compileDocs(ctx, targetBranch, delayedLibraryList, packageDir,
       excludeLibs, linkApi, postBuild);
 },
 description: 'Generate documentation for the provided libraries.',
 argParser: _dartDocParserConfig(targetBranch));
}
